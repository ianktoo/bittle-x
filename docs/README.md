# Bittle X Explorer — Documentation

Documentation for **Bittle X Explorer**, the web controller for the Petoi Bittle X robot dog.

## For users

- **[User guide](user-guide.md)** — Getting started, hardware, connection methods (BLE, USB, WiFi), controls (gamepad and AI), terminal usage, and troubleshooting.

The same content is available in the app: open the **Docs** (book) button in the header for the in-app documentation panel.

## For developers

- **[Development guide](development.md)** — Tech stack, repo structure, scripts, environment variables, services (Bluetooth, Serial, WiFi, Gemini), and build/deploy instructions.

## Reference

- **[OpenCat protocol](opencat-protocol.md)** — Command reference for the in-app terminal and skill buttons (gaits, postures, behaviors, servos, system commands).
